Key Features of the New Structure:

1. Image Optimization Strategy:
   - All images have WebP versions with PNG/JPG fallbacks
   - Images are organized by context (content, logos, team)
   - Original high-res versions preserved in separate directories
   - Thumbnails generated for performance
   
2. Performance Optimizations:
   - Critical CSS inlined in head
   - Non-critical CSS loaded asynchronously
   - JavaScript modules for code splitting
   - Image loading optimized with lazy loading
   
3. SEO & Accessibility:
   - Semantic HTML structure
   - Structured data implementation
   - ARIA attributes for improved accessibility
   - Meta tags and Open Graph data
   
4. Development Workflow:
   - Component-based architecture
   - Modular CSS and JavaScript
   - Clear separation of concerns
   - Easy to maintain and scale
