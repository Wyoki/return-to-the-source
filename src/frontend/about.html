Five Core Values

Non-negotiable principles that guide every lab, partnership and peso spent.

01

Memory Infrastructure

Oral testimony, sacred geographies and embodied ritual are treated as primary sources—equal to any colonial ledger.

Impact: Landscape becomes archive; song becomes data
02

Care is Infrastructure

A registered psychological counsellor is budgeted as line-item #1. Engaging genocide or slavery without a containment plan is unethical filmmaking.

Impact: Healing integrated into every process
03

African Frames First

European partners adapt to Namibian protocols—never the reverse. Agenda, language and tempo are set by the host community.

Impact: True narrative sovereignty
04

Process = Product

We green-light experiences, not projects. If transformation happens but no film emerges, the lab is still scored as a success.

Impact: Focus on transformative journey
05

Rotating Geographies

Each edition anchors in a different trauma landscape—building a lattice of Pan-African memory sites rather than a single branded campus.

Impact: Cross-continental knowledge exchange

Strategic Implementation

How our values translate into concrete action through four operational levers.

Memory Infrastructure

Values 1 & 5

We train filmmakers to treat oral testimony, sacred geographies and embodied ritual as primary sources equal to any colonial ledger. Each rotating geography becomes both classroom and archive, with local knowledge systems informing our pedagogical approach.

Trauma-Informed Pedagogy

Value 2

A registered psychological counsellor is embedded in every residency, ensuring that confronting genocide, slavery or forced migration happens inside a container of care. Mental health support isn't an afterthought—it's foundational infrastructure.

Community-Led Partnerships

Values 3 & 4

Rather than pitch-training extractive models, we unlock patient capital (Mastercard Enablement, Goethe-Institut, DW Akademie) that values cultural sovereignty and transformative process as outcomes in themselves.

Pan-African Network

Value 5

Our rotating geographies create cross-pollination between trauma landscapes, building solidarity networks that span continents. Madagascar's healing practices inform Namibian approaches; Cape Verdean musical traditions travel to Ethiopian sound archives.

Measuring What Matters

Success metrics that honor process over product.

Community Ownership

Percentage of curricula authored by local knowledge holders rather than external consultants

Healing Integration

Hours of psychological support provided per participant across all programming

Cross-Geography Exchange

Number of collaborative projects spanning multiple editions and trauma landscapes

Narrative Sovereignty

Stories told by and for African communities, measured against extraction-based metrics
